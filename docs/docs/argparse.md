## Argparse

`argparse` was a small command‑line argument parsing helper that existed in earlier
versions of this project. In the current codebase it is not part of the public
API, but the navigation entry is kept for compatibility with older revisions.

### Status

- **Available headers**: none in this version (`argparse.h` is not installed)
- **Recommended usage**: use your own CLI parser or a dedicated library

If you are browsing an older commit that still contains `argparse`, refer to that
revision for the exact API and examples.
