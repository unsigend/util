## Configparse

`configparse` was a lightweight configuration file helper used in earlier
iterations of this project. The current public API no longer ships this
module, but the documentation entry remains for historical context.

### Status

- **Available headers**: none in this version (`configparse.h` is not installed)
- **Recommended usage**: use your own configuration reader or an external library

If you are working with an older revision that still includes `configparse`,
please consult that revision for the detailed API and usage examples.
